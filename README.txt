This purpose of this project was creating a Airline Reservation system using .NET WEBAPI and ReactJS.
 database: MS SQL Server
Features of Project:
	1)Registration
	2)Login
	3)Searching Flights
	4)Booking a Flight
	5)Passanger Details
	6)Manage the Ticket----> a)Cancel Ticket   b)Reschedule Ticket  c)Edit Passanger details
	7)User Profile Update
	8)Ticket History of USer
